## 🌟 Fitur Utama & Keunggulan

<CardGrid stagger>
	<Card title="🤖 Multi-Provider AI Engine" icon="rocket">
		Mendukung **OpenAI**, **Google Gemini**, **Anthropic**, **DeepSeek**, **Ollama Local AI**, & Custom endpoints. Terkonfigurasi langsung dari `.env` tanpa dependensi luar yang berat.
	</Card>

	<Card title="📦 Paket NPM SDK Resmi" icon="setting">
		Siap terintegrasi dengan paket **`ai-biawak-sdk`** untuk pengintegrasian AI universal dan **`biawak-sdk`** untuk koneksi WebSocket client real-time & streaming AI.
	</Card>

	<Card title="⚡ Performa Super Cepat" icon="bolt">
		Ditenagai oleh **Bun v1.3+** runtime dan **Hono v4** web framework yang mampu menangani puluhan ribu request per detik dengan latensi minimal.
	</Card>

	<Card title="🗄️ Zero-Config SQLite → MySQL" icon="random">
		Langsung jalan tanpa install database server menggunakan **SQLite** (`sqlite.db`). Cukup ubah `DB_DRIVER=mysql` di `.env` untuk beralih ke MySQL saat production!
	</Card>

	<Card title="🔐 Hybrid Authentication" icon="shieldCheck">
		Mendukung **Better Auth** session cookies serta **JWT Bearer Token** (Access Token 15 mnt + Refresh Token 7 hari auto-rotation) & Google OAuth Sign-In.
	</Card>

	<Card title="📡 Real-Time WebSockets & BullMQ" icon="laptop">
		Dukungan native WebSockets (`/ws`) dengan streaming token AI real-time serta pemrosesan task antrean background **BullMQ + Redis**.
	</Card>
</CardGrid>
