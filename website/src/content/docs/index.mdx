## 🌟 Fitur Utama & Keunggulan

<CardGrid stagger>
	<Card title="⚡ Performa Super Cepat" icon="rocket">
		Ditenagai oleh **Bun v1.3+** runtime dan **Hono v4** web framework yang mampu menangani puluhan ribu request per detik dengan latensi minimal.
	</Card>

	<Card title="🗄️ Zero-Config SQLite → MySQL" icon="setting">
		Langsung jalan tanpa install database server menggunakan **SQLite** (`sqlite.db`). Cukup ubah `DB_DRIVER=mysql` di `.env` untuk beralih ke MySQL saat production!
	</Card>

	<Card title="🔐 Hybrid Authentication" icon="shieldCheck">
		Mendukung **Better Auth** session cookies serta **JWT Bearer Token** (Access Token 15 mnt + Refresh Token 7 hari auto-rotation) & Google OAuth Sign-In.
	</Card>

	<Card title="🦎 Biawak Code Scaffolder" icon="pencil">
		Fitur CLI bawaan `bun run make:route`, `make:middleware`, dan `make:schema` terisolasi di `node_modules` untuk mempercepat development.
	</Card>

	<Card title="📡 Real-time WebSockets & BullMQ" icon="laptop">
		Dukungan native WebSockets (`/ws`) serta pemrosesan task antrean background **BullMQ + Redis** dengan aman.
	</Card>

	<Card title="📄 Interactive Swagger UI & OpenAPI 3.0" icon="open-book">
		Dokumentasi API interaktif otomatis di `/api/doc` dan skema OpenAPI 3.0 JSON di `/api/openapi`.
	</Card>
</CardGrid>
