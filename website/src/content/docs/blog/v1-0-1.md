Kami sangat senang mengumumkan rilis **Biawak v1.0.1**! Pembaruan ini berfokus pada peningkatan stabilitas, pengalaman pengembang (Developer Experience), serta otomatisasi rilis paket NPM.

---

## 🌟 Apa yang Baru di v1.0.1?

### 1. ⚡ Generator Kode Terisolasi (`bin/biawak-make.js`)
Perintah `bun run make:route`, `make:middleware`, dan `make:schema` kini berjalan melalui binary executable yang terisolasi di `node_modules`. Pengguna mendapatkan pengalaman pembuatan berkas instan tanpa berkas skrip tambahan yang mengotori folder proyek.

### 2. 🗄️ Dukungan Kueri SQLite & Graceful Shutdown
- Menyesuaikan kueri health check `/api/health` agar mendukung `db.run()` pada driver `bun:sqlite` serta `db.execute()` pada driver `mysql2`.
- Memperbaiki `setupGracefulShutdown` saat menghentikan server dalam mode SQLite tanpa koneksi pool eksternal.

### 3. 📦 Pengoptimalan Ukuran Paket NPM
Menambahkan berkas `.npmignore` resmi untuk mengabaikan database lokal `sqlite.db`, berkas `.env`, dan log runtime. Ukuran terkompresi paket NPM `create-biawak-app` berkurang drastis dari **80 kB** menjadi **~15 kB**!

---

### 🚀 Cara Upgrade Proyek Anda:

```bash
# Menggunakan Bun
bun update create-biawak-app

# Atau menggunakan NPM
npm install create-biawak-app@latest
```
