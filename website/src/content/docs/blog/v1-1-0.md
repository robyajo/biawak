## 🦎 Apa yang Baru di Biawak v1.1.0?

Biawak Framework `v1.1.0` membawa pembaruan besar dengan menghadirkan **Multi-Provider AI Engine** yang siap pakai dan terkonfigurasi secara terpusat dari berkas `.env`, serta peluncuran dua paket NPM resmi: `ai-biawak-sdk` dan `biawak-sdk`.

### 🚀 Fitur Baru Utama (`feat`)

1. **Multi-Model AI Engine Terkonfigurasi via `.env`**:
   - Mendukung OpenAI (`gpt-4o-mini`, `gpt-4o`), Google Gemini (`gemini-1.5-flash`), Anthropic (`claude-3-5-sonnet`), DeepSeek (`deepseek-chat`), Ollama Local AI (`llama3`), dan Custom OpenAI-compatible endpoints.
   - Tanpa ketergantungan paket luar berat (zero-dependency fetch native adapter).
   - Helper server instan: `import { ai } from "@/ai"`.

2. **Ready-to-Use AI REST Endpoints**:
   - `POST /api/ai/chat`: Respon JSON non-streaming.
   - `POST /api/ai/stream`: Real-time SSE (Server-Sent Events) streaming.
   - `GET /api/ai/models`: Status keaktifan dan informasi provider.

3. **Real-Time AI Streaming over WebSocket**:
   - Engine WebSocket Biawak kini secara langsung menangani streaming token AI real-time via event WebSocket `ai:stream`.

4. **Paket NPM Resmi `ai-biawak-sdk`**:
   - SDK universal untuk mengintegrasikan model AI pada aplikasi client/server Node.js dan Bun (`npm install ai-biawak-sdk`).

5. **Paket NPM Resmi `biawak-sdk` (Client WebSocket SDK)**:
   - SDK client JavaScript/TypeScript untuk browser & Node yang mendukung koneksi WebSocket, auto-reconnect otomatis, pub/sub, dan streaming AI WebSocket (`npm install biawak-sdk`).
