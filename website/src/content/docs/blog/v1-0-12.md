## 🦎 Apa yang Baru di Biawak v1.0.12?

- 🚀 **Pembaruan & Perbaikan**: Peningkatan performa dan pemeliharaan arsitektur.
- 🛡️ **Stabilitas & Keamanan**: Pembaruan dependensi dan optimasi runtime Bun + Hono.

### 🐛 Perbaikan Bug & Optimasi
- **CLI Schema Generator (`make:schema`)**:
  - Memperbaiki isu di mana skema baru yang digenerate oleh `biawak-make` tidak otomatis terdaftar/diekspor dari file index skema.
  - Menambahkan **deteksi driver database aktif via `.env`**. Generator sekarang hanya akan membuat skema untuk database aktif yang terkonfigurasi (misalnya hanya membuat skema MySQL jika `DB_DRIVER=mysql`), memudahkan transisi total ke satu jenis database.
- **Database Reset Script (`db:reset`)**:
  - Mengubah pemuatan koneksi database ke **Dynamic Import** untuk menghindari error lock file `SQLITE_IOERR_VNODE` saat menghapus berkas SQLite.
  - Menambahkan **Programmatic Migration** otomatis saat mereset database. Dengan begitu, tabel database akan langsung dibangun ulang secara otomatis sebelum proses *seeding* berjalan, mencegah error `no such table` pada command `db:reset --seed`.
