## 🦎 Apa yang Baru di Biawak v1.0.12?

### 🐛 Perbaikan Bug & Optimasi
- **CLI Schema Generator (`make:schema`)**: Memperbaiki isu di mana skema baru yang digenerate oleh `biawak-make` tidak otomatis terdaftar/diekspor dari file index skema (`index.ts` di sqlite, mysql, dan wrapper utama). Kini skema baru akan langsung di-export secara otomatis sehingga terdeteksi oleh `drizzle-kit generate`.

# Menggunakan Bun
bun update create-biawak-app@latest
